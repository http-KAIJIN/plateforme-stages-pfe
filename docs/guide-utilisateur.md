# Guide Utilisateur - Stagio

Plateforme intelligente de gestion des stages et PFE.

## Connexion

1. Ouvrir le frontend.
2. Creer un compte ou se connecter.
3. La plateforme redirige automatiquement selon le role.

## Etudiant

- Consulter les offres dans `Stages`.
- Ouvrir une offre et cliquer sur `Postuler`.
- Suivre les statuts dans `Mes candidatures`.
- Consulter son PFE et deposer des rapports dans `Mon PFE`.
- Tester la compatibilite CV/offre dans `Analyse CV`.

## Entreprise

- Creer une offre dans `Mes offres`.
- Consulter les candidatures recues.
- Accepter ou refuser une candidature.

## Encadrant

- Consulter les PFE suivis.
- Ajouter des commentaires.
- Valider un PFE.

## Administrateur

- Voir les statistiques globales.
- Gerer les utilisateurs.
- Superviser stages et PFE.

## Notifications

La cloche dans la navbar affiche les notifications non lues. Cliquer sur une notification la marque comme lue.
